# 🛡️ Clearing Patterns (Goalie Initiated)

**🎯 Overview:** Structured clearing lanes for the defense to transition from a save to offensive possession.

## ⚙️ Setup
- Goalie inside the crease.
- Defensemen (D1, D2, D3) and Midfielders (M1, M2) in clearing lanes.

## 🏃 Execution
1. **The Break:** On a save, all players 'Break' to the sidelines.
2. **The Wing Pass:** Goalie looks to the Wing (Defenseman) first for a safe outlet.
3. **The Up-Pass:** Wing looks up-field to the Midfielder breaking across the midline.

### 🗺️ Visual Map
```text
    [ MIDLINE ]
     /   |   \n   (M1) (M2) (M3)
     |   ^   |
     |   |   |
   (D1) (G) (D2)
    [ CREASE ]
```

## 🧠 Mindset
- **Handle with Care:** The clear is the most vulnerable part of the game. Catch first, look second.
- **Communicate:** Goalie must give 'Clear' or 'Hold' commands.

## ✅ Outcomes
- Successful exit from the defensive zone under pressure.
- Quick-strike transition opportunities.
