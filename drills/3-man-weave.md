# 🏃 3-Man Weave (Full Field)

**🎯 Overview:** A high-speed passing drill to develop catching in stride and transition accuracy.

## ⚙️ Setup
- Three lines at the endline (Left, Center, Right).
- Ball starts with the Center player.

## 🏃 Execution
1. Center passes to the wing (Left or Right).
2. After passing, Center **MUST** run behind the player they passed to.
3. Receivers continue the pattern: pass, then run behind the receiver.
4. Drill finishes with a high-percentage shot on the far cage.

### 🗺️ Visual Map
```text
[L]      [C]      [R]
 |        |        |
 |   <----1----    |
 (C runs behind L) |
 |        |        |
 |   -----2---->   |
 | (L runs behind R)|
```

## 🧠 Mindset
- **Lead the Pass:** Don't throw to where they are; throw to where they are going.
- **Call Names:** Communication reduces dropped balls.

## ✅ Desired Outcomes
- Improved full-speed stick handling.
- Reduced transition unforced errors.

## 📋 Coaching Points
- Ensure players are following their pass (running behind).
- Use tennis balls if they are dropping too many reps to emphasize soft hands.
