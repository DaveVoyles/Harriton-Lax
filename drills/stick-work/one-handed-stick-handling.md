# 🥍 One-Handed Stick Handling Drill

**Video Reference:** [One-Handed Stick Handling | First Class Lacrosse](https://www.youtube.com/watch?v=7YYHks-Vexs)

## 🎯 Overview

One-handed stick work is essential for maintaining control during contact and developing a feel for the ball. This drill builds forearm strength and wrist flexibility.

**Target Audience:** All Players.
**Focus Area:** Stick Control and Forearm Strength.

## 💡 Key Concepts

1. **Leverage:** Using the midpoint of the stick to balance the weight of the head.
2. **Wrist Snap:** Using the wrist to generate power and control the ball's movement.
3. **Ball Retention:** Keeping the ball in the sweet
