# 🥍 1v1 Drill

**Video Reference:** N/A

## 🎯 Overview

The 1v1 drill is the cornerstone of individual player development. It isolates the fundamental matchup between a dodger and a defender, building habits that translate directly to settled and transition situations. This drill develops offensive creativity, defensive discipline, and the mental toughness required to win individual battles.

**Target Audience:** All Players (Attack, Midfield, Defense)
**Focus Area:** Dodging, On-Ball Defense, Footwork, Decision Making

## 💡 Purpose

1. **Offensive Development:** Train dodgers to execute split dodges, roll dodges, and face dodges at game speed from multiple field positions, building a full repertoire per [Dodging Essentials Basics](../offense/dodging-essentials-basics.md).
2. **Defensive Development:** Train defenders in proper approach angles, breakdown steps, body positioning, and recovery per [On-Ball Defense Mechanics](../defense/on-ball-defense-mechanics.md).
3. **Competitive Intensity:** Create a daily environment where players compete 1v1 with consequences, building mental toughness and accountability.

## ⚙️ Setup

- **Players:** Minimum 6 (3 offense, 3 defense). Can scale to full team with multiple stations.
- **Equipment:** Full pads, sticks, one goal, one goalie, cones for starting positions.
- **Field Layout:** Three starting positions rotate throughout the drill.

```
              ┌─────────────────────────────┐
              │           GOAL              │
              │          ┌───┐              │
              │          │ G │              │
              │          └───┘              │
              │            │                │
              │     X ─ ─ ─ ─ ─ ─ X        │
              │   (Behind/X)  (Behind/X)    │
              │            │                │
              │            │                │
  WING LEFT   │            │                │  WING RIGHT
  ┌───┐       │            │                │       ┌───┐
  │ D │ ←─────│────────────│────────────────│─────→ │ D │
  │ O │       │            │                │       │ O │

              │            │                │
              │            │                │

              │         │ TOP │             │
              │         │ D O │             │
              │         └─────┘             │
              └─────────────────────────────┘

  D = Defender starting position
  O = Offensive player starting position
  G = Goalie
```

- **Starting Position 1 — Wing:** Dodger starts at the wing (restraining line extended), defender is 5 yards inside. Dodger attacks toward the goal from the alley.
- **Starting Position 2 — Top Center:** Dodger starts at top center (12 yards above the crease), defender is between the dodger and the goal. Classic north-south dodge lane.
- **Starting Position 3 — X (Behind Goal):** Dodger starts behind the goal at GLE extended. Defender sets up in a trail position. Dodger must work around GLE to create a scoring opportunity.

## 🏃 Execution

### Phase 1: Wing 1v1s (5 min)

1. Offensive player starts at the wing with the ball. Defender is in an on-ball stance 5 yards toward the cage.
2. On the whistle, dodger attacks. He must choose a dodge: split, roll, or face.
3. Defender executes proper approach — inside hip, stick on gloves, choppy feet into a breakdown.
4. Play is live until a shot, a caused turnover, or the ball goes out of bounds.
5. Rotate offense and defense after each rep. Every player gets 2 reps from each side (left wing and right wing).

### Phase 2: Top 1v1s (5 min)

1. Dodger starts at top center. Defender is squared up between the dodger and the crease.
2. Dodger must attack north-south — no drifting laterally past the hash marks.
3. This position favors the split dodge and the question-mark dodge. Encourage the dodger to attack the front hip of the defender.
4. Defender must turn and recover if beaten; no reaching or slashing.

### Phase 3: X / Behind Goal 1v1s (5 min)

1. Dodger starts behind the cage at X. Defender is in a trail position at GLE.
2. Dodger works to get topside around GLE using a roll dodge, inside roll, or sweep.
3. Defender must maintain inside position — staying between the dodger and the crease at all times.
4. If the dodger gets topside, the defender must recover hip-to-hip and force a contested shot.

### Phase 4: King of the Hill (5 min)

1. Winner stays on. Dodger who scores stays on offense; defender who gets a stop stays on defense.
2. Rotate starting positions each rep.
3. First player to 3 wins is the "King" and sits out while the next challenger comes in.

## 🧠 What Players Can Expect

- **Intensity 🔥:** This drill is physically demanding. Every rep is a full-speed competitive battle. Expect to be gassed after 3 consecutive reps.
- **Communication 🗣️:** Defenders should be vocal — "I've got ball!" on every rep. Goalies call out screens and positioning: "You're good!", "Stay inside!"
- **Failure is Learning 📐:** Dodgers will get stopped. Defenders will get beat. The purpose is to develop counters. If your split dodge gets stuffed, set up the face dodge next rep.

## ✅ Desired Outcomes

- Dodgers develop a go-to dodge AND a counter move from each field position.
- Defenders build confidence in their on-ball technique and learn to recover when beaten.
- Both sides improve conditioning and mental toughness under competitive pressure.
- Goalies get high-quality, game-realistic save opportunities from multiple angles.

## 📋 Coaching Points

- **Dodgers — Sell the First Move:** The dodge doesn't work if you telegraph it. Use your eyes, feet, and stick position to set up the defender before attacking. Review [Dodging Essentials Basics](../offense/dodging-essentials-basics.md).
- **Defenders — Don't Lunge:** The most common mistake is lunging at the ball carrier. Stay in your stance, keep your feet moving, and play the hands. If beaten, sprint to recover — don't swipe. Review [On-Ball Defense Mechanics](../defense/on-ball-defense-mechanics.md).
- **Short-Stick vs. Long-Pole Variation:** Run a round where short-stick middies defend against dodgers. This simulates real game matchups and forces middies to use body position over stick length.
- **Time-Limited Reps:** Add a 6-second clock. If the dodger hasn't gotten a shot off in 6 seconds, the defense wins. Builds urgency and decisiveness.

---

_Note: This drill is a daily staple. Run it at every practice to build competitive habits._
