# 🏹 4-Point Shooting Drill

**🎯 Overview:** Rapid-fire shooting from the primary dodger locations (Top-Left, Top-Right, Wing-Left, Wing-Right).

## ⚙️ Setup
- 4 balls placed at the 'shorthorn' positions.
- One shooter starts in the 'Hole' (crease area).

## 🏃 Execution
1. Shooter sprints to Point 1 (Top Left), collects, and dodges/shoots.
2. Immediately reset to the hole and sprint to Point 2 (Top Right).
3. Repeat for all 4 positions.

### 🗺️ Visual Map
```text
        [ CAGE ]
           ||
           ||
   (3)----[HOLE]----(4)
    |       |        |
    |       |        |
   (1)------|-------(2)
```

## ✅ Desired Outcomes
- Conditioning while maintaining shooting form.
- High-rep accuracy from all angles.
